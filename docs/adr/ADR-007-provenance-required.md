# ADR-007: Provenance required for knowledge records

- **Status:** Accepted
- **Date:** 2026-07-19

## Context

Users and agents must know where knowledge came from and when it was validated.

## Decision

Require source metadata (type, provider, reference, model, reviewer, verification/validation timestamps) where applicable.

## Consequences

Better auditability and safer AI retrieval. Slightly more authoring friction.
